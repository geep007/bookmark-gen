# MCP Service

This directory will contain the MCP (Model Context Protocol) integration services for syncing bookmarks from Twitter and LinkedIn.

## Planned Implementation

### Twitter Sync (tweetmash MCP)
- Connect to tweetmash MCP connector
- Fetch all Twitter bookmarks
- Parse and normalize bookmark data
- Detect duplicates before insertion

### LinkedIn Sync (linkedmash MCP)
- Connect to linkedmash MCP connector
- Fetch all LinkedIn bookmarks
- Parse and normalize bookmark data
- Detect duplicates before insertion

## Epic 1, Story 1.3 and 1.4

These services implement the requirements from:
- Story 1.3: MCP Integration - Twitter Bookmark Sync
- Story 1.4: MCP Integration - LinkedIn Bookmark Sync
